# ECUT-AutoLogin (东华理工大学校园网自动登录工具)

ecut-autologin-jiangxifuzhou这是一个专为 **东华理工大学 (ECUT)** 学生编写的校园网自动登录脚本。它可以帮助你在电脑开机或断网时，一键完成“先注销旧连接、再登录新账号”的操作，支持多运营商后缀切换。

## 🛠️ 运行环境准备

在使用此脚本前，你的 Windows 电脑需要完成以下两步配置：

### 1. 安装 Python 环境
保证目前电脑有python环境

### 2. 安装 requests 运行库 (核心组件)
脚本需要使用 `requests` 工具来发送网络请求。请打开 **PowerShell** 或 **命令提示符 (CMD)**，输入以下命令安装：

```powershell
python -m pip install requests
```
> **注意**：如果安装时提示权限问题，请尝试以管理员身份运行 PowerShell。

## 🚀 快速开始

1. **下载脚本**：下载本仓库中的 `main.py` 文件到本地。
2. **配置账号**：右键点击 `main.py` 选择“编辑” (或用记事本打开)，修改顶部的配置：
   ```python
   USERNAME = "学号@fzcmcc"  # 替换为你的学号，记得保留后缀
   PASSWORD = "你的密码"      # 替换为你的校园网密码
   ```
3. **运行脚本**：在文件夹内点击右键，选择“在终端中打开”，输入：
   ```powershell
   python main.py
   ```

## 🌐 后缀参考指南
请根据你的宽带办理情况修改 `USERNAME`：
* **中国移动**：`@fzcmcc`
* **中国电信**：`@telecom`
* **中国联通**：`@unicom`
* **校园网自办**：不带后缀 或 自行抓包确认
